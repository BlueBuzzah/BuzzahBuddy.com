# BuzzahBuddy.com

Landing page for the BuzzahBuddy companion app.

## About

BuzzahBuddy is a mobile companion app for BlueBuzzah tactile gloves, providing wireless Bluetooth LE control and session tracking for Parkinson's patients. This repository contains the static "coming soon" landing page for the app.

## Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling via CDN
- **Vanilla JavaScript** - Minimal interactivity

No build process required. The site is fully static and can be deployed anywhere.
